# AI Agent Project

This project features an AI agent designed to assist with various tasks, primarily focusing on file system operations and Python script execution. The agent can perform the following actions:

- **List files and directories**: Explore the project structure.
- **Read file contents**: View the content of any file.
- **Execute Python files**: Run Python scripts with optional arguments.
- **Write or overwrite files**: Create new files or modify existing ones.

## How to Use

Interact with the AI agent by providing commands and queries related to file management and Python script execution. The agent will interpret your requests and utilize its capabilities to perform the desired actions.
